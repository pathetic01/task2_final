Формула для круга в треугольнике: (radius <= Math.sqrt(3) * side_b / 6).
Где:
radius - Радиус основания окружности;
Math.sqrt(3) - квадратный корень из 3;
side_b - сторона треугольника;
формулу взял с этого сайта:
